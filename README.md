# Schoolmeal Frontend
학교 정보와 급식을 교육청에서 가져와 표시합니다.

### Development
    git clone https://github.com/effx13/schoolmeal-frontend.git
    cd schoolmeal-frontend/
    yarn install
    yarn start
### Build
    git clone https://github.com/effx13/BubblePopServer.git
    cd BubblePopServer/
    yarn install
    yarn build
